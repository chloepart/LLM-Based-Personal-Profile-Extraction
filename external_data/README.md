# External Data

This folder contains external datasets and downloaded data:

## Structure

- **senate_html/** - Downloaded HTML profiles of U.S. Senators
  - Used by senate extraction pipeline
  - Downloaded from official Senate website

## Notes

These files are typically:
- Large (excluded from version control)
- Downloaded or generated externally
- Referenced by experiment scripts

If you need to re-download or regenerate data, see `../docs/USAGE.md` for instructions.
